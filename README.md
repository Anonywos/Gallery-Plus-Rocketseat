# 📸 Gallery Plus

Esse projeto é o segundo feito como parte do curso de React da plataforma Rocketseat. O objetivo do projeto é criar uma galeria de fotos com capacidade de criar, salvar, filtrar e deletar fotos e álbuns usando fundamentos de React TypeScript. A aplicação é capaz de conversar com rotas do backend, rotas essas que já vem com o esqueleto do projeto, fornecido pela Rocketseat.

## 🚀 Tecnologias

- React
- TypeScript
- CSS / TailwindCSS
- Vite
- HTML
- React Router
- localStorage
- Tailwind Variants (TV)
- React Query
- React Form
- Sonner
- nuqs
- zod

## 🎯 Funcionalidades

- Listar e visualizar fotos vindas do backend
- Criar novas fotos e salvá-las no backend
- Filtrar fotos por nome e por álbum
- Criar novos álbuns e selecionar fotos para adicioná-los
- Visualizar detalhes da imagem
- Navegar entre fotos
- Excluir fotos
- Atualizar os álbuns de uma foto

## ⚙️ Como rodar

```bash
git clone https://github.com/Anonywos/Gallery-Plus-Rocketseat.git
cd Gallery-Plus-Rocketseat
npm install
npm run dev
```

## 🔐 Variáveis de ambiente

Crie um arquivo `.env` na raiz do projeto com base no exemplo abaixo:

```env
VITE_API_URL=http://localhost:XXXX
VITE_IMAGES_URL=http://localhost:XXXX/images
```

### Descrição

- `VITE_API_URL`: URL base da API backend
- `VITE_IMAGES_URL`: URL base para acesso às imagens
